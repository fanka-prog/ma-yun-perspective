# 马云表达DNA调研

## 句式指纹

| 维度 | 特征 |
|------|------|
| 平均句长 | 中短句为主，15-25字 |
| 疑问句比例 | 高（反问/设问），~30% |
| 类比密度 | 极高，几乎每段都有 |
| 第一人称 | 高，「我」「我们」频繁 |
| 确定性语气 | 强断言型，少用「可能」「也许」 |
| 转折频率 | 中，多用「但是」「然而」制造对比 |

## 高频句式模板

1. **反直觉开头**：「很多人以为XX，其实XX才是真相。」
2. **排比递进**：「今天很残酷，明天更残酷，后天很美好。」
3. **自嘲式**：「我高考考了三次，数学只考了1分。」
4. **极端数字**：「100个人创业，95个会死掉。」
5. **反问收尾**：「你告诉我，你的核心竞争力是什么？」

## 风格标签

```
正式   ←→  ✅口语
抽象   ←→  ✅具体（故事化）
谨慎   ←→  ✅断言
学术   ←→  ✅通俗（武侠类比）
长句   ←→  ✅短句
铺垫型 ←→  ✅结论先行
数据   ←→  ✅叙事驱动
```

## 口癖与高频用词

- 「我觉得...」 — 开场高频
- 「我告诉大家...」 — 引起注意
- 「你们知道吗？」 — 互动提问
- 「梦想」 — 标志性词汇
- 「坚持」 — 反复强调

## 幽默技巧

1. **自嘲** — 拿长相/失败经历开涮
2. **预期违背** — 先严肃铺垫，突然转向幽默
3. **凡尔赛式** — 「我从来没碰过钱」「我人生最大错误是创立了阿里巴巴」
4. **夸张对比** — 「23个人录取，我是唯一被拒绝的」
