# 马云重大决策与行动记录

> 调研时间：2026年6月

## 关键决策时间线

### 1. 阿里巴巴创立（1999）
- **决策**：B2B模式，服务中小企业
- **团队**：18人凑50万，湖畔花园公寓
- **逻辑**：中国中小企业缺乏出口渠道

### 2. 淘宝创立+免费对抗eBay（2003）
- **决策**：秘密研发6人封闭开发
- **核心策略**：免费三年对抗eBay收费
- **追加**：2005年再投10亿继续免费
- **结果**：2006年淘宝占C2C 70%，eBay退出中国
- **关键词**：「免费是最贵的」

### 3. 支付宝拆分（2011）
- **决策**：从阿里集团拆分支付宝独立运营
- **争议**：股东不满（Yahoo/SoftBank认为违约）
- **马云逻辑**：合规优先+用户利益优先
- **事后评价**：「我没有后悔」

### 4. 传承计划（2019退休）
- **决策**：55岁退休，张勇接班
- **背景**：「10年前开始规划接班人」
- **核心理念**：公司要靠系统和文化，不是靠个人

### 5. 阿里重组（2023-2025）
- **背景**：精简层级、聚焦一线、重拾创业精神
- **状态**：马云无正式头衔，幕后方向性指导

## 决策风格总结

| 特征 | 表现 |
|------|------|
| 快速决策 | 不拖沓，淘宝秘密开发仅1个月 |
| 敢于争议 | 支付宝拆分、免费策略均引发巨大争议 |
| 长期视角 | 传承计划提前10年；102年愿景 |
| 价值观优先 | 多次在利润和价值观冲突时选价值观 |
